# WIPO 商标风险排查 — 使用教学

> 本文档面向同事，教你如何用两阶段流程排查产品文案中的商标风险词汇。

## 为什么要查商标？

产品名、包装文案中的英文词汇如果与目标市场（美国、加拿大、德国、法国、英国）已注册的玩具类（Nice 28类）商标完全一致，可能存在侵权风险。上线前需要排查。

---

## Part A：用网页工具查大部分词

### 第1步：打开网页工具

在浏览器中打开：

> https://wipo-webapp-g6pgwo9sau82gvjkegsxxr.streamlit.app/

### 第2步：输入要查的文本

在页面顶部的文本框中，粘贴你的产品文案或命名方案。

例如：
```
We are launching a new toy line called MagicBlocks and HappyBuilder.
The box says "Build Fun, Robot Kids".
```

工具会自动从文本中提取所有候选词和词组（英文/法文/德文），包括：
- 单词（≥3字母，自动过滤 the/a/is 等停用词）
- 连续词组（2-4词，如 "magic blocks"、"build fun"）

**💡 换行技巧**：词组只在同一行内组合，换行可以分隔不同短语。例如分行输入：
```
Hello Kitty
Outdoor Toy
```
只会查 "hello kitty" 和 "outdoor toy"，不会跨行组合出 "kitty outdoor"。如果不想生成词组，每个词单独一行即可。

### 第3步：设置参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 请求间隔 | 2-3秒（默认2秒） | 并发查5个办公室，间隔太短容易触发风控。词多时建议调到3秒 |
| IP办公室 | US, CA, DE, FR, GB（默认） | 5个目标市场全选 |
| Nice分类 | 28（默认） | 玩具类 |
| 商标状态 | Registered（默认） | 只看已注册商标 |

### 第4步：开始排查

点击「开始排查」按钮。工具会：
1. 连接 WIPO 商标数据库
2. 逐个查询每个候选词在5个办公室的注册情况
3. 实时显示进度（当前第几个词、查到什么）

大约每词 6-10 秒（5个办公室并发 + 认证重试 + 间隔等待）。建议单次候选词控制在 **15 个左右**，约需 2-3 分钟，耗时预估最准确。候选词超过 25 个时实际耗时会明显变长（约为预估的 2 倍），且更容易触发风控。

### 第5步：查看结果

排查完成后，页面会显示：

- **风险词汇** — 与已注册商标完全一致的词，列出商标名、办公室、注册号
- **失败词条** — 因网络风控（403）未能查到的词

> 只有「完全一致」才算风险。比如你查 "camp"，商标 "CAMPING" 不会匹配，只有商标名就叫 "CAMP" 的才算。

### 第6步：一键重查失败词（如有）

如果页面出现「一键重查失败词」按钮，说明有些词因为云端网络风控没查到。

**点击该按钮**，工具会：
- 自动对失败词重新查询
- 间隔多加1秒（降低风控概率）
- 重查只允许1次

重查完成后，页面会显示绿色或黄色提示，明确告诉你结果：

| 提示颜色 | 含义 |
|---------|------|
| 🟢 绿色「已全部查全，均未发现风险商标」| 失败词都查完了，没有风险 ✅ |
| 🟢 绿色「已全部查全，新增 X 个风险词」| 失败词都查完了，还发现了新的风险词，已加到下方报告中 ✅ |
| 🟡 黄色「仍有 X 个词条未查全，已查到的词条均未发现风险」| 还是有词没查到，但查到的都没有风险 |
| 🟡 黄色「仍有 X 个词条未查全，已查到的 X 个风险词已整合到报告中」| 还是有词没查到，但查到的部分发现了新风险词，已加到下方报告中 |

> 不管哪种情况，**已查到的结果都会自动整合到下方报告中**，你不需要做额外操作。

### 第7步：复制失败词列表（如重查后仍有失败）

如果重查后仍有失败词，页面会显示：
- 警告提示：「建议用飞书助手本地查询」
- 一个可复制的词条列表

**复制这个列表**，然后进入 Part B 用飞书助手查。

---

## Part B：用飞书助手查失败词

### 为什么需要飞书助手？

网页工具运行在云端服务器上，高频查询时容易被 WIPO 风控限制（返回403）。而飞书助手连接的是你本地的 WorkBuddy，使用本地网络查询，**几乎不会触发403**。

### 第1步：打开飞书找「商标风险排查助手」

在飞书中搜索并打开 **商标风险排查助手**（WorkBuddy 应用）。

### 第2步：发送失败词

把 Part A 复制的失败词列表直接粘贴发送给助手。例如：

```
帮我把这些词查一下商标风险：
hello kitty
build fun
robot kids
```

或者一行一个词也可以：

```
hello
kitty
build fun
robot
```

### 第3步：等待查询结果

助手会自动：
1. 提取你发送的词条
2. 逐个查询 WIPO 商标数据库（5个办公室并发）
3. 在对话中直接回复查询结果

因为用的是本地网络，查询过程很稳定，基本不会出现403。

### 第4步：查看结果

助手会在对话中返回一份 Markdown 格式的报告，包含：
- 风险词汇列表（商标名、办公室、注册号）
- 如有失败词会单独列出（概率极低）

## 常见问题

### Q: 文本多长合适？
**A:** 建议单次候选词控制在 **15 个左右**（约 100-200 字文案），此时预估耗时与实际最接近（约 2-3 分钟），403 风险也最低。候选词超过 25 个时，实际耗时会明显变长（约为预估的 2 倍），且更容易触发风控。长文本请分段多次查。

### Q: 词组会被查吗？
**A:** 会。工具自动提取 2-4 词的连续词组。比如 "Build Fun" 会拆成 build、fun、build fun 三个词分别查。**词组只在同一行内生成**，换行可以避免不相邻的词被组合。比如分行输入 "Hello Kitty / Outdoor Toy" 只会查 hello kitty 和 outdoor toy，不会查 kitty outdoor。

### Q: 法语/德语文本能查吗？
**A:** 能。工具支持英/法/德三种语言，自动归一化重音字符（é→e, ü→u）。

### Q: 只查美国可以吗？
**A:** 可以。在网页工具的 IP办公室 设置中只选 US 即可。

### Q: 查到的风险词怎么办？
**A:** 建议更换产品名或修改文案，避免使用与已注册商标完全一致的词汇。如果认为属于合理使用，请咨询法务。

### Q: 网页工具打不开？
**A:** Streamlit Cloud 偶尔会休眠，首次打开可能需要等 30 秒唤醒。如果持续打不开，直接用飞书助手查全部词也可以。
